Optional bufferSize

bufferSize: number
  • Defined in json-rx/JsonRxClient.ts:30

Number of messages to keep in buffer before sending them to the server. The buffer is flushed when the message reaches this limit or when the buffering time has reached the time specified in bufferTime parameter. Defaults to 100 messages.

Optional bufferTime

bufferTime: number
  • Defined in json-rx/JsonRxClient.ts:36

Time in milliseconds for how long to buffer messages before sending them to the server. Defaults to 10 milliseconds.
