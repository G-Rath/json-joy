Module json-pack

JSON Pack

Library for encoding and decoding JavaScript native structures to MessagePack format.

Use Encoder to encode plain JSON values.

import {Encoder, Decoder} from 'json-joy/{lib,es6,esm}/json-pack';

const encoder = new Encoder();
const decoder = new Decoder();
const buffer = encoder.encode({foo: 'bar'});
const obj = decoder.decode(buffer);

console.log(obj); // { foo: 'bar' }

For more:

  • Use Encoder to encode only JSON values.
  • Use EncoderFull to also encode binary data, extensions and pre-computed MessagePack buffers.
  • To encode binary data use Uint8Array.
  • To encode an extension use JsonPackExtension.
  • To encode a pre-computed MessagePack value use JsonPackValue.
